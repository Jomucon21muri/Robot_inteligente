## Descripción General del Proyecto

La construcción de un robot humanoide es un proceso complejo que requiere una combinación de habilidades mecánicas, electrónicas y de programación. Este proyecto integra:

- **Diseño mecánico**: Esqueleto resistente que soporta peso y permite movimiento
- **Componentes electrónicos**: Sensores y motores para percepción y movimiento
- **Programación**: Software para control, reconocimiento de objetos y ejecución de tareas
- **Inteligencia Artificial**: Capacidades de aprendizaje y adaptación

## Objetivos del Proyecto

1. **Construir** un robot humanoide funcional desde cero
2. **Integrar** sistemas mecánicos, electrónicos y de software
3. **Implementar** capacidades de IA para tareas autónomas
4. **Desarrollar** investigación aplicable a doctorado en robótica

## Herramientas Necesarias

### Software
- CAD: SolidWorks, Fusion 360, FreeCAD
- Programación: Python, C++, ROS
- Simulación: Gazebo, V-REP, PyBullet
- Control de versiones: Git

### Hardware
- Impresora 3D / Mecanizado CNC
- Microcontroladores: Arduino, Raspberry Pi, Jetson Nano
- Sensores: Cámaras, LIDAR, IMU, sensores de fuerza
- Actuadores: Servomotores, motores paso a paso
